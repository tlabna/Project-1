The Basics of the Web and HTML

How the web works

The web is a bunch of computers that communicate with each other. When a person surfs the The World Wide Web (www), their computer sends a HTTP request (Hypertext Transfer Protocol) to a server which responds with an appropriate html document. Your web browser then interprets the document and displays it on the screen.

HTML

HTML stands for Hypertext Markup Language. HTML documents contain text content which describes "what you see" and markup which describes "how it looks".


HTML Vocabulary

Tag: An HTML tag is always contained within angled brackets. Most tags have an opening tag (< p > for example) and a closing tag, (< /p >). Some tags (called void tags) do not require a closing tag (like the < br > tag).

Element: An HTML element refers to everything within a set of opening and closing tags.

Attribute: This is a property of an HTML element. For example, to set the href attribute of an anchor tag to the Udacity URL, you would write < a href="www.udacity.com" >. Here's the attribute in action: Udacity Website

Inline vs. Block: HTML elements are either inline or block. Block elements form an "invisible box" around the content inside of them.
