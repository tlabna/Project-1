Lesson 1: The Basics of the Web and HTML

How the web works

The web is a bunch of computers that communicate with each other. When a person surfs the The World Wide Web (www), their computer sends a HTTP request (Hypertext Transfer Protocol) to a server which responds with an appropriate html document. Your web browser then interprets the document and displays it on the screen.

HTML

HTML stands for Hypertext Markup Language. HTML documents contain text content which describes "what you see" and markup which describes "how it looks".

HTML Vocabulary

Tag: An HTML tag is always contained within angled brackets. Most tags have an opening tag (< p > for example) and a closing tag, (< /p >) Some tags (called void tags) do not require a closing tag (like the < br > tag).

Element: An HTML element refers to everything within a set of opening and closing tags.

Attribute: This is a property of an HTML element. For example, to set the href attribute of an anchor tag to the Udacity URL, you would write < a href="www.udacity.com" >. Here's the attribute in action: Udacity Website

Inline vs. Block: HTML elements are either inline or block. Block elements form an "invisible box" around the content inside of them.

Lession 2: Creating a Structured Document with HTML

The Tree-Like Structure of HTML

The "tree-like structure" comes from the fact that HTML elements can have other elements inside of them. When experts talk about the DOM (Document Object Model), they are talking about the tree-like structure of an HTML page. In a browser, this structure shows up as a series of nested boxes.

Developer Tools

Browsers come with developer tools where you can inspect elements on webpages. Not all elements are displayed but you can see the whole structure (DOM) of a webpage. You can also play around with styles on webpages and see changes in real-time using these developer tools.

Text Editors

Programmers use special text editors to code. These editors make the programmer's life easier. For example, some text editors will automatically generate a closing HTML tag when you write an opening tag. Sublime Text is an example of a great text editor. Lesson 3: Adding CSS Style to HTML Structure

What is CSS?

CSS stands for Cascading Style Sheets and is a style sheet language used for describing the look and formatting of an HTML document written in a markup language. This is done by giving similar HTML elements the same class name and then specifying the style that should apply to that class.

Avoiding Repetition

As a programmer it is very important to avoid repetition. Here are some reasons why:

  • Avoiding Errors: Rewriting the same code over and over can lead to errors when the programmer decides to make a change to something.
  • Consistency
